Getting Started

Class used to navigate object structures.

Package Managers

npm: npm install walker.js

bower: bower install walker.js

Walker

Class used to interactively navigate through an object structure. Walk active keys or search for a specific node.

Class
Walker(opts)

Parameters

Name Type Default Description
opts object Walker options.
opts
Name Type Default Description
seperator string '.' Seperator used for tracking location.
location string ' ' Current location. By default at root. 
var walker = new Walker({
  seperator : '.',
  location  : ''
});

each

Execute a specified iterator for each node in an object.

Function
each(obj, iterator)

Parameters

Name Type Description
obj object Object to traverse.
iterator function Function called for each iteration. The function will be passed two arguments, `location` and `value`. 
walker.each({
  'walk': { 'each': 'node' }
}, function (locaton, val) {
  console.log(location);
  console.log(val); 
  console.log('--');
});

// walk
// { 'each': 'node' }
// --
// walk.each
// node
// --

bubble

Execute a specified iterator for each parent node of the specified query key. Callback will be passed a string representing the location and the value at that location.

Function
bubble(obj, query, iterator)

Parameters

Name Type Description
obj object Object to traverse.
query string Formatted string representing the property at which to begin bubbling from.
iterator function Function executed at each level. 
walker.bubble({
  'bubble': { 'from': { 'here': 'value' } }
}, 'bubble.from.here', function (locaton, val) {
  console.log(location);
  console.log(val);
  console.log('--');
});

// bubble.from.here
// value
// --
// bubble.from
// { 'here': 'value' }
// --
// bubble
// { 'from': { 'here': 'value' } }
// --

search

An interface to grab a specified node based on a query string. When a value in the namespace if not found there are two options. By default undefined will be returned. If you set the third parameter, `create`, to true an empty object will be created at that node and walker will continue to traverse the object tree.

Function
search(obj, query, create)

Parameters

Name Type Description
obj object The object to search on.
query string Formatted string representing a key in the obj.
create object Flag for if we should create an empty object when an undefined property is found. 
var val = walker.search({
  'nested': { 'node': 'value' }
}, 'nested.node');

console.log(val);
// value
